If your browser is able to support WebGL, the game should appear on the left side of the screen. If you would like to use another version, please use the links below. The PC and Mac downloads must be unzipped to play. The executable is named Squawker.

Security Warnings

As the executable does not come from the App or Windows Store, you will most likely receive a security warning message.

MacOS

If you receive this warning on MacOS, you need to go to System Preferences then choose Security & Privacy. Under the General tab, at the bottom of the window you should see a message saying Squawker was blocked from use because it's not from an identified developer. Clicking the Open Anyway button should allow you to run the game.

Windows

On Windows, if you see a "Windows Protected Your PC" message with no option to run the program, click the More Info link. This should reveal a button that says "Run Anyway" that will allow you to run the game.

Post Game Survey

When you are finished playing the game, please go back to the original window where you completed the survey in order to take the post-game survey.
